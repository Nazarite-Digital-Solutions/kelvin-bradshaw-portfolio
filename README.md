# Kelvin Bradshaw | Personal Portfolio

A modern, responsive personal portfolio website showcasing my experience in customer service, front-end web development, and digital solutions.

The website was designed and developed using HTML, CSS, and JavaScript with a strong focus on responsive design, accessibility, performance, and clean user experience.

---

## 🌐 Live Website

https://kelvin-bradshaw.vercel.app/

---

## 📖 About

This portfolio serves as a central hub for recruiters and potential employers to learn more about me, explore my projects, view my qualifications, and get in touch.

One of the key features is **NOVA**, an interactive voice assistant that allows visitors to navigate the website and ask questions about my experience using speech recognition and text-to-speech technology.
